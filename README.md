# beat_sales

b:eat 프랜차이즈 랜딩페이지 관련 저장소입니다.

## 구성

```
b-eat-landing/
├── s0.html              # 랜딩페이지 S0(히어로) 섹션 HTML
└── assets/
    ├── hri-2-photo.png  # 히어로 배경 사진
    └── logo.png         # b:eat 로고
```

## 진행 상황

- [x] S0 (히어로) 섹션
- [ ] 나머지 섹션 (통계, 가격표, 클라이언트 로고, 갤러리, 문의폼 등)

## 출처

Figma 디자인 파일을 기반으로 추출 및 변환되었습니다.
https://www.figma.com/design/LXp3YnfekN9uhQVVYDHi92/b-eat-프랜차이즈-랜딩페이지

## 실행 방법

`b-eat-landing/s0.html` 파일을 브라우저에서 직접 열면 됩니다. `assets/` 폴더와 같은 위치에 있어야 이미지가 정상적으로 표시됩니다.
